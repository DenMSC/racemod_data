Racemod Menu
------------

- Automatically show intro menu when new player connects to the server.
  Intro menu lets you set basic binds and select if you want to use the recommended settings.

- Ingame "gametypemenu" with hud options / binds / game options.
  There are also some useful commands as well as some help.
  Hud options include special options set by huds/<hudname>.hudoptions file.

- Ingame help menu, loads from http://livesow.net/race/ui/help_pages (can be viewed in browser)
  Loads from the website so it's easier to change/add things.
